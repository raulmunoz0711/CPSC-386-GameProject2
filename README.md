## This is the GitHub for my CPSC 386 game project 2. This is to ensure I have version control as I continue my progress on this assignment.

I am making a Maze Game that contains 3 levels. To complete each maze, you must first grab all the keys that are in the level. There will be a total of 6 game scenes. Main Menu, Settings, game1, game2, game3, and level selection. Main Menu will be in charge of playing the background music and allowing the user to navigate to the Settings scene, quit the game, and open the level selection scene.. The Settings scene will allow the user to adjust the volume of the background music playing, and a chance to quit the game. The level selection scene, was made with the use of AI and is in charge of ensuring that each level remained locked until the user complete the level before(User cant play level 2 without beating level 1). Finally, each game represents each level, the 3 game scenes represent 3 levels each increasing in difficultly.
