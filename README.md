## This is the GitHub for my CPSC 386 game project 2. This is to ensure I have version control as I continue my progress on this assignment.

I am making a Maze Game that contains 3 levels. To complete each maze, you must first grab all the keys that are in the level. The number of keys will correspond to the number of the level. Ex: Level 1 has 1 key, Level 2 has 2 keys, etc. There will be a total of 5 game scenes. Main Menu, Settings, game1, game2, and game3. Main Menu will be in charge of playing the background music and allowing the user to navigate to the Settings scene and quit the game. The Settings scene will allow the user to adjust the volume of the background music playing. Finally, each game represents each level.
